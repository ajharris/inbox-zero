# Gmail Inbox Manager Frontend

This project is a React application that allows users to manage their Gmail inbox by marking all emails as read and scrolling through messages at a user-defined speed to exempt specific files.

## Features

- Authenticate with Google to access Gmail.
- Mark all unread emails as read.
- Scroll through emails at a user-defined speed.
- Exempt specific emails from being marked as read.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node package manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd gmail-inbox-manager/frontend
   ```

2. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```
This will open the application in your default web browser at `http://localhost:3000`.

### Building for Production

To create a production build of the application, run:
```
npm run build
```
This will generate a `build` folder containing the optimized application.

## Usage

Once the application is running, you can authenticate with your Google account and manage your inbox. Use the controls to set the scrolling speed and exempt specific emails.
